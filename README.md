**KrishiMitra**

KrishiMitra is a website for farmers that includes an interactive chatbot system and information pages for crops, fertilizers, and government schemes. The chatbot provides predefined 
guidance to help farmers with common queries.  

Features
- Interactive Chatbot: Users can type questions about crops like wheat, rice, sugarcane, and fertilizers like urea, DAP, etc., and get instant predefined answers.
- Crop Guidance Pages:Detailed information about crop care, irrigation, and growth stages.
- Fertilizer Guide Pages: Details about common fertilizers and their usage.
- Schemes Pages: Information about government schemes for farmers.
- Responsive Design: Works well on both desktop and mobile devices.

# How to Use
1. Open index.html in a browser.
2. Navigate to the chatbot to ask queries about crops or fertilizers.
3. Browse Crop, Fertilizer, and Schemes pages for additional information.

# Technologies Used
- HTML, CSS, JavaScript

# Notes
- The chatbot only provides predefined answers for crops and fertilizers.
- Schemes pages contain static information and do not provide automatic responses.
